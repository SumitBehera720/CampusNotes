# CampusNotes

A premium, modern web application for university students to share, discover, and collaborate on academic notes. Built with Python and Flask.

## Features
- **Student Dashboard**: Track downloads, saves, and recent interactions.
- **Social System**: Follow other students and get notified about new uploads.
- **Gamification**: Earn badges and climb the leaderboard by contributing high-quality notes.
- **Access Control**: Secure note viewing, downloading, and bookmarking.

*Built for students, by students.*
